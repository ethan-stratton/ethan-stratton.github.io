IGN Debanding

← strattao · lab

Color banding is the ugly staircase pattern you see in gradients, glows, fog, and dark scenes — anywhere colors transition slowly across few bits of precision.

Interleaved Gradient Noise is one line of shader code that kills it. It adds a tiny imperceptible noise pattern that breaks up the bands. Unlike random noise (which looks grainy), IGN is spatially coherent — neighboring pixels get similar-but-different offsets, so the result looks smooth to your eye.

From Acerola's video: must be applied at draw time, not as post-processing. The holy grail for debanding glows and fog in 2D games.

The Code

This is literally the entire thing. One function.

// Interleaved Gradient Noise — Jorge Jimenez, 2014
float IGN(vec2 screenPos) {
  return fract(
    52.9829189 * fract(
      dot(screenPos, vec2(0.06711056, 0.00583715))
    )
  );
}

// Usage: add to color before quantization
color += (IGN(gl_FragCoord.xy) - 0.5) / levels;
