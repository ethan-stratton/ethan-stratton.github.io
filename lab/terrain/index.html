How It Works

A 2D heightmap is generated using fractional Brownian motion (fBm) — multiple octaves of Perlin noise layered at increasing frequencies and decreasing amplitudes. Low octaves create the broad continental shapes; high octaves add craggy detail.

Lacunarity controls how quickly frequency increases per octave (higher = more detail variation). Persistence controls how quickly amplitude decreases (higher = rougher terrain, lower = smoother).

The heightmap is rendered as an isometric column grid. Each cell is projected into 2D isometric space, then extruded downward based on its height value. Columns are drawn back-to-front for correct occlusion. Faces are shaded — top faces get full color, side faces are darkened to sell the 3D illusion.

Colors are mapped from height using gradient stops — each preset defines a different palette that evokes a specific alien biome. The water plane is a flat layer drawn at a configurable threshold, with slight transparency.

The same Perlin noise from the Liquid Noise demo powers the heightmap generation. Same math, completely different application.
