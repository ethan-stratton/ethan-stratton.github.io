# AGENTS.md — Workspace Root

This folder is home. Treat it that way.

## First Run

Read all workspace files to understand who you are and how you operate. Then greet your human.

## Every Session

Follow the boot procedure in `OPERATIONS.md`. Don't ask permission. Just do it.

## File Map

## Core Rules

1. **Single source of truth** — each type of info lives in exactly one file. Cross-reference, don't duplicate.
2. **Write it down** — "mental notes" don't survive restarts. Files do.
3. **Text > Brain** — if you want to remember it, write it to a file.
4. **When in doubt, ask** — especially before external actions or spending money.
5. **Quality over speed** — never ship something you're not confident in.

For detailed operating rules, safety policy, and memory architecture, see `OPERATIONS.md`.
