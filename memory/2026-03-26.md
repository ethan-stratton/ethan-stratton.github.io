## Light System Fix — 20:42-20:48 UTC

### Root cause of dim lights
1. Gradient texture alpha was quadratic (a*a) — with Additive blend (SourceAlpha), double-attenuated to quartic falloff. Fixed: alpha=1f in gradient, only RGB fades.

3. Bumped lantern radius 120→180, rocket glow 200→280.

## Light System Rewrite — 21:00-22:42 UTC

### Multiply light map approach (attempt 2)

### Corrected to proper Terraria/Starbound technique (attempt 3)
- Researched: standard approach is BLACK light map + ADDITIVE lights + MULTIPLY over scene
- Light map cleared to BLACK (no light), not ambient color
- Ambient base: inverse of darkness overlay (day=white=visible, night=dim)
- Light sources drawn ADDITIVELY (they stack naturally with ambient)
- Multiply light map over scene: scene × lightmap
- Premultiplied alpha gradient: linear falloff, Color(rgb, rgb, rgb, alpha)
- Still needs visual confirmation from Ethan
