# 2026-03-16

## Session: Arena Shooter Movement System

Big session building out the movement system. Added:
- **Ropes** — climb up/down, crouch to hold position, jump/cartwheel off, double-tap S to drop, per-rope lengths
- **Dash** — double-tap A/D for faster run speed while held, ground only
- **Wall climbing** — left wall with gap for enemies, vault animation to top, wall hop (jump + press toward wall to reattach), tri-directional aim away from wall
- **Vault kick** — jump during slide for flying kick with hurtbox, speed boost, i-frames. Requires fresh space press (combo feel), minimum slide time before cancel
- **Settings menu** — Esc to toggle: enemies, slide, cartwheel, dash, double jump, wall climb, rope climb, drop through, vault kick, music, quit
- **BGM** — Pantera Floods Outro (slowed/reverbed), toggleable in menu
- Enemies default to OFF now
- Crouch-slides chainable (no lockout)
