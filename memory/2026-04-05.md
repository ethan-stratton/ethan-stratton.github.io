### Lighting Polish Round 2 (19:15-19:45 UTC)

**Torch banding root cause:** Single gradient layer on 50% ambient only adds ~20-30% brightness — not enough dynamic range for smooth falloff. 8-bit color shows visible steps in the narrow transition. Fix: triple-stacked layers per torch (outer warm 280px + middle white 187px + core white 93px) pushes center to ~1.9× ambient.

**Gradient texture evolution:** 256px → 512px → 1024px. Added IGN dithering (from Acerola notes). Changed from quadratic `(1-d²)²` to Gaussian `exp(-d²×6)`. Composite sampler changed PointClamp → LinearClamp. All needed but the triple-stack was the real fix.

**Lava noise surface:** Reused water's `DrawScrollingNoiseWater` approach — scissor-clipped tiling noise with LinearWrap. Key lessons:
- Must be fully opaque (checkerboard floor showing through looked terrible)
- Same-scale noise layers create visible tiling seam lines at full opacity
- Fix: 3 layers at different scales (5×, 3×, 2×) break each other's seam patterns
